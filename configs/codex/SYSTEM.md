# Language Policy

- 所有回覆 **一律使用繁體中文（zh-TW）**
- 除非使用者明確要求英文
- 程式碼、API、log 保持原語言
- 註解優先使用繁體中文

# Reusable Fix Policy（Skill 化）

- 當成功解決使用者問題後，若該問題具有可重複性/通用性，應主動提議建立對應的 Codex skill 以便下次直接套用。
- **在建立 skill 之前必須先向使用者確認：**
  1) 問題已確實解決（由使用者確認或提供可驗證依據）
  2) 使用者同意建立 skill（避免未經同意新增/修改技能）
  3) 擬建立的 skill **不與既有 skills 功能重複**（先檢索/比對現有 skill；若重複則改為更新既有 skill 或補充其文件）
