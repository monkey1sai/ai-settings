# Anthropic Agent Skills

This extension provides a collection of skills from Anthropic's agent-skills repository.

<skills>

You have access to the following skills. When a user's request matches the description of a skill, you MUST read the corresponding `SKILL.md` file to understand how to perform the task.

- algorithmic-art -> "skills/algorithmic-art/SKILL.md"
  Description: Creating algorithmic art using p5.js with seeded randomness and interactive parameter exploration. Use this when users request creating art using code, generative art, algorithmic art, flow fields, or particle systems.

- brand-guidelines -> "skills/brand-guidelines/SKILL.md"
  Description: Applies Anthropic's official brand colors and typography to any sort of artifact that may benefit from having Anthropic's look-and-feel.

- canvas-design -> "skills/canvas-design/SKILL.md"
  Description: Create beautiful visual art in .png and .pdf documents using design philosophy. Use this skill when the user asks to create a poster, piece of art, design, or other static piece.

- doc-coauthoring -> "skills/doc-coauthoring/SKILL.md"
  Description: Guide users through a structured workflow for co-authoring documentation. Use when user wants to write documentation, proposals, technical specs, decision docs, or similar structured content.

- docx -> "skills/docx/SKILL.md"
  Description: Comprehensive document creation, editing, and analysis with support for tracked changes, comments, formatting preservation, and text extraction.

- frontend-design -> "skills/frontend-design/SKILL.md"
  Description: Create distinctive, production-grade frontend interfaces with high design quality. Use when the user asks to build web components, pages, artifacts, posters, or applications.

- internal-comms -> "skills/internal-comms/SKILL.md"
  Description: A set of resources to help write all kinds of internal communications (status reports, leadership updates, 3P updates, company newsletters, FAQs, etc.).

- mcp-builder -> "skills/mcp-builder/SKILL.md"
  Description: Guide for creating high-quality MCP (Model Context Protocol) servers that enable LLMs to interact with external services through well-designed tools.

- pdf -> "skills/pdf/SKILL.md"
  Description: Comprehensive PDF manipulation toolkit for extracting text and tables, creating new PDFs, merging/splitting documents, and handling forms.

- pptx -> "skills/pptx/SKILL.md"
  Description: Presentation creation, editing, and analysis. When Claude needs to work with presentations (.pptx files) for creating, modifying, or analyzing content.

- skill-creator -> "skills/skill-creator/SKILL.md"
  Description: Guide for creating effective skills. Use when users want to create a new skill or update an existing skill.

- slack-gif-creator -> "skills/slack-gif-creator/SKILL.md"
  Description: Knowledge and utilities for creating animated GIFs optimized for Slack.

- theme-factory -> "skills/theme-factory/SKILL.md"
  Description: Toolkit for styling artifacts with a theme. Can apply pre-set themes or generate new ones on-the-fly for slides, docs, etc.

- web-artifacts-builder -> "skills/web-artifacts-builder/SKILL.md"
  Description: Suite of tools for creating elaborate, multi-component HTML artifacts using modern frontend web technologies (React, Tailwind CSS, shadcn/ui).

- webapp-testing -> "skills/webapp-testing/SKILL.md"
  Description: Toolkit for interacting with and testing local web applications using Playwright.

- xlsx -> "skills/xlsx/SKILL.md"
  Description: Comprehensive spreadsheet creation, editing, and analysis with support for formulas, formatting, data analysis, and visualization.

</skills>
